Easing (all actions except Text)

Set an easing factor for mouse movements. The higher this value is, the more will mouse movements seem “natural” or “human-like”, which also implies: will be slower. If this option is used, the actual speed will also depend on the distance between the start and the end position, i.e. the time needed for moving will be higher if the distance is larger.
