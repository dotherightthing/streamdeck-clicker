DISPLAYS
Load Displays info

This feature uses macOS AppleScript to get information about your System Display dimensions.

This information can be useful when choosing X-Y Points.

Clicking “Load Display info will display a prompt from macOS, asking you to give “Elgato Stream Deck.app” access to control “Finder.app”.

At the prompt, click “OK” if you trust this plugin and wish to run an AppleScript to query your System Display dimensions. That will also hide the button and this message.

Displays info

Width

Width of the selected display in System Settings > Displays.

Height

Height of the selected display in System Settings > Displays.

Horz Offset

Horizontal offset of selected display, relative to the main display in System Settings > Displays > Arrange Displays.

Vert Offset

Vertical offset of selected display, relative to the main display in System Settings > Displays > Arrange Displays.

FIRST X-Y POINT
First X-Y Point info

Screen location at which to perform the action.

On a Mac, use COMMAND+SHIFT+4 to reveal crosshairs with X-Y position.

At the correct position, make a note of the X (top) and Y (bottom) values.

Press ESC to hide the crosshairs again.

Enter the X and Y values into the fields above.

Adjust the Wait and Easing options when testing different X and Y values.

SECOND X-Y POINT
Second X-Y Point info

Screen location at which to perform a second action (optional).

OPTIONS
Easing info

Slow down mouse movements to make them seem more 'natural' or 'human-like'.

The actual speed will depend on the distance between the start and end positions.

Wait info

Number of milliseconds to wait after clicking.

Increase this value to test that you have the correct X-Y position.

Restore info

Restore initial mouse location after action.
