FIRST X-Y POINT
First X-Y Point info

Screen location at which to perform the click action.

Reference the DISPLAY BOUNDS when deciding whether to use positive or negative values.

Adjust the Wait and Easing options when testing different X and Y values.

SECOND X-Y POINT
Second X-Y Point info

Screen location at which to perform a second click action (optional).

DISPLAY BOUNDS
Load Display Bounds info

This feature uses macOS AppleScript to get information about your System Display dimensions.

This information can be useful when choosing X-Y Points.

Clicking “Load Display Bounds“ will display a prompt from macOS, asking you to give “Elgato Stream Deck.app” access to control “Finder.app”.

At the prompt, click “OK” if you trust this plugin and wish to run an AppleScript to query your System Display dimensions. That will also hide the button and this message.

Display Bounds info

Top

Top boundary of highest display in System Settings > Displays > Arrange Displays.

A negative value indicates the vertical offset of a second display, arranged above the main display.

Left

Left boundary of all displays in System Settings > Displays > Arrange Displays.

A negative value indicates the width of a second display, arranged to the left of the main display.

Width

Width of the main display in System Settings > Displays.

Height

Combined height of all displays in System Settings > Displays.

OPTIONS
Easing info

Slow down mouse movements to make them seem more 'natural' or 'human-like'.

The actual speed will depend on the distance between the start and end positions.

Wait info

Number of milliseconds to wait after clicking.

Increase this value to test that you have the correct X-Y position.

Restore info

Restore initial mouse location after action.
